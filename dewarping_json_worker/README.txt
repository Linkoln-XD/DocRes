Скрипт mat_to_npy использовать прямо в файле, в конце есть две переменные с путями.
Входная директория с папками файлов .mat.
Выходная директория.
В выходной директории будет создана та же структура фалов, что и во входной, только с файлами с расширением .npy.

Скрипт tst запускать и использовать через GUI.
